## Portfolio Management Dashboard – Qlik Sense
# Project Overview

The Portfolio Management Dashboard is an interactive Qlik Sense BI solution designed to provide a comprehensive view of the company’s overall investment portfolio.
The dashboard enables stakeholders to monitor Assets Under Management (AUM), investments, redemptions, and customer profiles across multiple channels, categories, and time periods.

This solution helps management and business teams make data-driven decisions by offering a single consolidated view of portfolio performance and customer behavior.

## Business Objectives

Track the company’s overall portfolio performance

Analyze channel-wise investments, redemptions, and current AUM

Monitor customer category and profile-wise fund distribution

Identify trends using date-based analysis

Enable drill-down analysis for detailed customer insights

## Tools & Technologies

BI Tool: Qlik Sense

Data Source: Enterprise Database (Oracle / PostgreSQL)

Data Preparation: SQL

## Data & Reporting Workflow

Extracted portfolio, transaction, and customer data from the source database

Applied SQL transformations to ensure:

Data accuracy

Removal of duplicates

Consistent aggregations

Loaded cleansed data into Qlik Sense
Designed an interactive dashboard using:

KPIs

Filters

Drill-down tables

Enabled dynamic selections for real-time analysis
